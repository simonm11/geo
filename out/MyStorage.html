MyStorage

new MyStorage()

Utilisation de localStorage pour préserver les modifications dans le mode offline Exemple d'utilisation : > add('edit', {'osm_id' : 1, 'highway': 'secondary'}); > add('edit', {'osm_id' : 2, 'highway': 'primary'}); Indiquera que l'attribut 'highway' des objet 1 & 2 doivent être respectivement modifier en 'secondary' & 'primary' > get('edit'); returns : [{'osm_id' : 1, 'highway': 'secondary'}, {'osm_id' : 2, 'highway': 'primary'}]; A chaque lancement de l'application, ces modif seront appliqué aux objects OpenLayers. On peux ensuite enregistrer ces modification dans la base de données postgis > save();
